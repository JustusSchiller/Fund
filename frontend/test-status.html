功能测试清单:

投资功能:
1. 连接MetaMask钱包
2. 进入Projects页面
3. 点击项目的"Invest"按钮
4. 输入投资金额
5. 确认投资 → 应该弹出MetaMask确认窗口
捐款功能:
1. 连接MetaMask钱包
2. 进入Projects页面
3. 点击项目的"Donate"按钮
4. 输入捐款金额
5. 确认捐款 → 应该弹出MetaMask确认窗口
创建项目:
1. 连接MetaMask钱包
2. 进入Create Project页面
3. 填写项目信息
4. 点击"Create Project" → 应该弹出MetaMask确认窗口
5. 项目创建后应该出现在Projects列表中
DEX交易:
1. 连接MetaMask钱包
2. 进入DEX页面
3. 选择交易对（ETH/USDC等）
4. 输入交易金额
5. 点击"Execute Private Swap" → 应该弹出MetaMask确认窗口
问题报告:
如果遇到"Failed to create"或其他错误，请：
1. 刷新页面（Ctrl+F5）
2. 检查浏览器控制台（F12）
3. 确保MetaMask已安装且已连接
4. 确认网络连接正常
